# Simple Music Player - Android 5.1 Compatible
## Description
This is a brand new music player developed from scratch, specifically for Android 5.1 (API 22), perfectly compatible with 32-bit systems.
## Features
- minSdkVersion: 22 (supports Android 5.1)
- targetSdkVersion: 22 (perfectly compatible)
- Pure Java code, no native dependencies, supports 32-bit
- Supports playing local music files
- Background playback service
- Simple UI design
## How to Build
### Method 1: Online Build (Recommended)
1. Go to https://www.online-java.com/APKBuiler/
2. Upload the source code
3. Click build, download the APK directly
### Method 2: Android Studio Build
1. Open Android Studio
2. Create new project, replace source code with this
3. Click Build -> Generate Signed APK
4. Sign and install
## Usage
- Grant storage permission to play music on SD card
- Support play/pause control
- Support background playback
